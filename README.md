***nuxt-frontend-sandbox***

## About

Nuxt + TypeScript でのフロントエンド開発の実験場  
商品管理アプリ

*[Backend - springboot-api-sandbox](https://github.com/KotaTanaka/springboot-api-sandbox)*

## Technology

* **Nuxt.js** `v2`
* script - **TypeScript**
* template - **pug**
* style - **SCSS**

## Getting Started

* Install

```bash
$ yarn
```

* Run

```bash
$ yarn dev
```

* Build

```bash
$ yarn build
```

→ http://localhost:5050

* Open Storybook (Component Catalog)

```bash
$ yarn storybook
```

→ http://localhost:6006
