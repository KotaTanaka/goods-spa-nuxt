# goods-spa-nuxt

## About

商品管理SPA [Nuxt.js]

[バックエンドリポジトリ goods-api-kotlin](https://github.com/KotaTanaka/goods-api-kotlin)

## Technology

* Nuxt.js
* TypeScript

## Getting Started

* ソースコードのクローン

```
$ git clone git@github.com:KotaTanaka/goods-spa-nuxt.git
```

* 依存パッケージのインストール

```
$ cd goods-spa-nuxt
$ npm i
```

* アプリケーション起動

```
$ npm run dev
```

→ http://localhost:5050
