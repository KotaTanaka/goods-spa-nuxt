# goods-spa-nuxt

## About

商品管理SPA [Nuxt.js]

*[バックエンドリポジトリ goods-api-kotlin](https://github.com/KotaTanaka/goods-api-kotlin)*

## Technology

* **Nuxt.js**
* script - **TypeScript**
* template - **pug**
* style - **SCSS**

## Getting Started

* ソースコードのクローン

```
$ git clone git@github.com:KotaTanaka/goods-spa-nuxt.git
```

* 依存パッケージのインストール

```
$ cd goods-spa-nuxt
$ yarn
```

* アプリケーション起動

```
$ yarn dev
```

→ http://localhost:5050
